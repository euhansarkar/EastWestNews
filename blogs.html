.forEach(), .map(), .filter() .... What's the difference?

.forEach() method, is used to execute the same code on every element in an array but does not change the array and it returns undefined.

.map() method executes the same code on every element in an array and returns a new array with the updated elements.

.filter() method checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria.

.find() method checks every element in an array to see if it meets a certain criteria and returns only one value with the elements that return truthy for the criteria.
