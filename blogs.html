what is the difference between var, let and const in javascript?

Var, let and const are basically used for variable declaration in javascript. When you declare a variable you by using var, you can redeclare it. But you can't redaclare a variable if you are use let or const. By using var and let you can reassign a variable. But you can't reassign a variable if you declare it by using const. it is the simple difference.
